# NFT Marketplace Smart Contract

This repository contains a simple NFT marketplace smart contract built using Solidity and Hardhat.

## Features
- Mint NFT tokens
- List NFTs for sale
- Purchase listed NFTs
- Basic marketplace logic

## Tech Stack
- Solidity
- Hardhat
- OpenZeppelin Contracts

## Purpose
This project demonstrates basic smart contract structure for an NFT marketplace, including minting, listing and executing sales on-chain.
